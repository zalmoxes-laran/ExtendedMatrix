From SQLite to LibreOffice Base step-by-step (for windows):

1. Download and install the odbc driver from: http://ch-werner.de/sqliteodbc/
2. Open Command Prompt and insert: C:\Program Files\LibreOffice\program\odbcconfig.exe
3. Click on Add button in the DNS user tab.
4. Select SQLite3 ODBC Driver and click finish.
5. When the SQLite ODBC DSN Configuration window opens enter the Data Source Name: USM_ICCD, Browse to the database EMdb_USM-v.0.db, and click the OK button.
